Számábrázolási formák

Típus Leírás Példa
Egész szám (Integer) Pozitív és negatív egész számok ábrázolása. Általában 32 vagy 64 biten tárolják. int a = 42;
Lebegőpontos szám (Floating Point) Valós számok ábrázolása. Az IEEE 754 szabványt használják, például egyszeres (32 bit) vagy kétszeres pontosságú (64 bit). float b = 3.14f;
Fixpontos szám (Fixed Point) Valós számok ábrázolása, ahol az egész és törtrész külön tárolódik. Gyakran használják beágyazott rendszerekben. int c = 12345; // 123.45-nek felel meg, ha a tört rész 2 tizedesjegyű
Bináris (Binary) A számok 2-es számrendszerben való ábrázolása. Az informatikában alapvető jelentőségű. 0b1010 // 10 decimálisan
Hexadecimális (Hexadecimal) A számok 16-os számrendszerben való ábrázolása. Gyakran használják a memória címzésében és színkódokban. 0x1A3F // 6719 decimálisan
Oktális (Octal) A számok 8-as számrendszerben való ábrázolása. Régebbi rendszereknél és bizonyos alkalmazásoknál használatos. 0755 // 493 decimálisan
BCD (Binary-Coded Decimal) Olyan számábrázolási forma, ahol minden decimális számjegyet négy bináris bit ábrázol. 0011 1001 (39 decimálisan)
